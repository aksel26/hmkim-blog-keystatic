## 1. MDOC 파일 생성 로직 제거
- MDOC 파일 생성 관련 모든 코드를 삭제해주세요
- 진행 단계(progress steps)에서도 MDOC 관련 단계를 제거해주세요

## 2. 초안 발행부터 검증까지 워크플로우 변경

### 새로운 워크플로우 순서
1. **AI 검토** → 2. **콘텐츠 개선** → 3. **검증** → 4. **사용자 검토**

### 상세 요구사항

#### 사용자 검토 단계
- 사용자가 피드백을 입력할 수 있는 UI 제공
- 피드백 기반 개선 내역을 사용자에게 표시
- **SEO 체크리스트와 점수를 Popover 컴포넌트로 표시**
  - 체크리스트 항목들 (예: 메타 설명, 키워드 밀도, 제목 최적화 등)
  - 각 항목별 점수 또는 통과/실패 상태
  - 전체 SEO 점수

#### 수정 요청 처리
- 사용자가 수정 요청 시, **AI 검토 단계부터 전체 워크플로우 재실행**
- 수정 이력 추적 및 표시

#### 최종 승인 단계
**승인(게시하기) 버튼:**
1. 클릭 시 확인 모달 표시
2. 모달에서 확인 버튼 클릭 시:
   - PR 생성 프로세스 시작
   - Loading 상태 UI로 표시
   - 완료 후 "수동으로 PR 승인이 필요합니다" 안내 메시지 표시

**보류하기 버튼:**
1. 클릭 시 작업 상태를 '보류'로 변경
2. `/jobs` 페이지로 자동 이동
3. 보류된 작업은 언제든지 다시 AI 초안 작성부터 재개 가능하도록 구현

## 4. 템플릿 적용 설명 문서 작성
Life와 Tech 카테고리 선택 시:
- 각 템플릿 옵션이 무엇인지
- 선택한 옵션에 따라 어떤 필드/구조가 적용되는지
- 템플릿별 차이점과 사용 사례

위 설명을 코드 주석 또는 별도 마크다운 문서로 작성해주세요.
